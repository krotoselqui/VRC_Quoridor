ボードゲーム『コリドール』の補助U#ギミックになります。
２人用になっています。

今後も更新を予定しております。


ギミックの内容(v0.91)

①板がスナップする
（板をはめ込む位置におさまる形になります）
②自分および相手の残り板の枚数が確認可能
③リセットが可能


導入方法

1.UnityのプロジェクトにVRCSDK3とUdonSharpを導入してください。
2.内容データのunitypackageをインポートしてください。
3.Quoridorフォルダ内にある"Qurodior.prefab"を
　ワールドにD&Dで設置してください。


注意事項など

・本データはVRChat SDK3のワールド(UdonSharp導入済状態)での利用のみを想定しています。それ以外の環境では動作しません。
